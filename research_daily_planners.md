# Research: Popular Daily Organizer/Planner Apps

## Common Features Across Top Daily Planner Apps

### Core Functionality
1. **Task Management**
   - Create, edit, delete tasks
   - Subtasks and checklists
   - Task prioritization (high/medium/low)
   - Due dates and deadlines
   - Recurring tasks

2. **Calendar Integration**
   - View tasks on calendar
   - Schedule events
   - Time blocking / timeboxing
   - Sync with Google Calendar, Outlook, Apple Calendar

3. **Organization Systems**
   - Projects/folders to group tasks
   - Tags and labels
   - Filters and search
   - Multiple views (list, board, calendar, timeline)

4. **Reminders & Notifications**
   - Push notifications for due tasks
   - Custom reminder times
   - Location-based reminders (some apps)

5. **Collaboration (team features)**
   - Share tasks/projects
   - Assign tasks to team members
   - Comments and attachments
   - Real-time sync

### Popular Apps Breakdown

**Todoist** - Most popular
- Simple, clean interface
- Natural language input ("tomorrow at 3pm")
- Productivity tracking (karma points, streaks)
- Filters and labels
- Cross-platform

**TickTick** - Feature-rich
- Calendar view built-in
- Pomodoro timer
- Habit tracking
- Eisenhower Matrix view
- Custom smart lists

**Motion** - AI-powered
- AI automatically schedules tasks
- Optimizes calendar based on priorities
- Reduces decision fatigue
- Auto-reschedules when things change

**Sunsama** - Daily planning focus
- Daily planning ritual
- Time blocking
- Review past days
- Integrates with other tools
- Focus on work-life balance

**Google Calendar** - Event scheduling
- Simple event creation
- Multiple calendars
- Sharing and collaboration
- Integration with Gmail

**Notion** - All-in-one workspace
- Fully customizable
- Databases, wikis, docs
- Templates
- Complex but powerful

**Trello** - Visual Kanban
- Card-based system
- Drag and drop
- Visual workflow
- Good for projects

## What These Apps DON'T Do

None of these apps provide:
- Energy level tracking
- Personality-based recommendations
- Optimal timing suggestions based on personal energy
- Biorhythm or circadian rhythm insights
- Success pattern analysis
- Correlation between energy and outcomes
- Predictive scheduling based on personal patterns
- Life path / numerology integration
- Weather/moon phase energy insights
- Personalized "best time" recommendations

## Key Insight

**Standard daily planners focus on WHAT and WHEN you do things.**
**They don't tell you WHEN IS BEST FOR YOU personally based on your energy patterns.**
